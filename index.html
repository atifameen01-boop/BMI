Medical Citations & Scientific References

  • Basal Metabolic Rate (BMR): Calculated using the Mifflin-St Jeor Equation (Mifflin et al., 1990, American Journal of Clinical Nutrition). Individuals with extreme adiposity are tailored using the lean mass-driven Katch-McArdle Formula.
  • Body Fat Percentage: Modeled using the U.S. Navy Circumference Equation (Hodgdon & Beckett, 1984, U.S. Navy Medical Technical Report). Classifications align with standards of the American Council on Exercise (ACE).
  • Physical Activity Level (PAL): Configured via standard multipliers or dynamically calculated using the peer-reviewed FAO/WHO/UNU 24-Hour Time-Allocation Model (Joint Expert Consultation on Human Energy Requirements, Rome, 2001/2004).
  • Metabolic Equivalents (METs): Energy expenditure estimations based on standard multipliers derived from the Compendium of Physical Activities (Ainsworth et al., 2011, Medicine & Science in Sports & Exercise).
  • Obesity Classifications & Waist Ratios: Standard thresholds and BMI divisions derived directly from guidelines published by the World Health Organization (WHO) and the International Diabetes Federation (IDF).

Google Sheets Setup Guide

Follow these simple steps to link this app directly to any Google Sheet:

  1. Create a brand new **Google Sheet** (or open an existing one).
  2. In the top menu, go to Extensions → Apps Script.
  3. Delete any placeholder code in the script editor and **paste the Apps Script Code** provided below.
  4. Click Deploy → New Deployment (top right).
  5. Set the select box under "Select type" to Web App (click the gear icon if you don't see it).
  6. Configure access settings exactly like this:
    • Execute as: Me (your-email)
    • Who has access: Anyone
  7. Click Deploy, authorize permissions, and **copy the Web App URL** displayed.
  8. Paste that copied URL in the URL input located in the settings panel above, then save configurations.
Apps Script Code Template 
function doPost(e) {
  try {
    var sessionData = JSON.parse(e.postData.contents);
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    
    // Set headers if the sheet is completely empty
    if (sheet.getLastRow() === 0) {
      sheet.appendRow([
        "Timestamp", "Subject Name", "Email ID", "Sex", "Age", "Goal", "Activity Level",
        "Height", "Weight", "BMI", "BMI Classification", "Body Fat %", "Body Fat Classification",
        "AS Index Value", "AS Index Classification", "BMR (kcal)", "TDEE (kcal)", "Calorie Target"
      ]);
    }

    // Append calculated diagnostic parameters as a new row
    sheet.appendRow([
      new Date().toLocaleString(),
      sessionData.subjectName || "N/A",
      sessionData.subjectEmail || "N/A",
      sessionData.sex || "N/A",
      sessionData.age || "",
      sessionData.goal || "N/A",
      sessionData.activity || "N/A",
      sessionData.height || "N/A",
      sessionData.weight || "N/A",
      sessionData.calculations?.bmi || "N/A",
      sessionData.calculations?.bmiClass || "N/A",
      sessionData.calculations?.bodyFatPercentage || "N/A",
      sessionData.calculations?.bodyFatClassification || "N/A",
      sessionData.calculations?.asIndex || "N/A",
      sessionData.calculations?.asIndexClassification || "N/A",
      sessionData.calculations?.bmr || "N/A",
      sessionData.calculations?.tdee || "N/A",
      sessionData.calculations?.calorieGoal || "N/A"
    ]);

    return ContentService.createTextOutput(JSON.stringify({ "status": "success" }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(error) {
    return ContentService.createTextOutput(JSON.stringify({ "status": "error", "error": error.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
